# WillCounter
A minimalist time‑tracking app built with Jetpack Compose. This app helps you track sessions, count days, and manage history — all with a clean dark UI.
# Features
- Circular Timer with animated progress bar.
- Tap the timer circle to start/stop instantly.
- Shows elapsed time (HH:MM:SS) and day count.
- Start/Restart button to reset and log sessions.
- Session history list with timestamps.
- Delete button (tomato red 🗑️) to clear history.
- Timer persists after app is closed using SharedPreferences.
- Modern dark theme UI with Material3.
